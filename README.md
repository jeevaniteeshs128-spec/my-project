# Full Pace Frontend Prototype

Frontend-only movie social platform built with React + TypeScript + Vite.

## Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Router
- shadcn-style UI components
- Lucide Icons

No backend, no database, and no API integration.
All screens use local mock data.

## Implemented Pages

1. Landing Page
2. Login Page (role-based portals)
3. Reviewer Dashboard
4. Director Dashboard
5. Community Dashboard
6. Movie Details Page
7. Movie Community Page
8. Diary Page
9. Watchlist Page
10. Chat Page
11. Meetups Page
12. Notifications Page
13. User Profile Page
14. Settings Page

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
